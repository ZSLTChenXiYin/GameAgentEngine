# GameAgentCreator Guide

[**中文**](./GUIDE_GAMEAGENTCREATOR.md) | **English**

Main packaged Creator pages:

- `Worlds`
- `Settings`
- `State`
- `Timelines`
- `Continuity`
- `Logs`
- `Traces`

If you want world time advancement, configure `world_time_settings` in `Settings` first, then inspect `world_time_state` and tick results in `State`, `Timelines`, and `Continuity`.
