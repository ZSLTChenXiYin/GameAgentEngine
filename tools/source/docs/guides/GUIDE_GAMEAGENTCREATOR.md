# GameAgentCreator 指南

**中文** | [**English**](./GUIDE_GAMEAGENTCREATOR_EN.md)

当前随包 Creator 的核心页面：

- `Worlds`
- `Settings`
- `State`
- `Timelines`
- `Continuity`
- `Logs`
- `Traces`

如果你要使用世界时间推进，请先在 `Settings` 配置 `world_time_settings`，然后在 `State`、`Timelines`、`Continuity` 查看 `world_time_state` 和推进结果。
