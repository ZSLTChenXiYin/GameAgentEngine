# GameAgentCreator 指南

**中文** | [**English**](./GUIDE_GAMEAGENTCREATOR_EN.md)

GameAgentCreator 是 GameAgentEngine v0.2.0 的 Web 可视化编辑器。它通过浏览器与引擎 HTTP API 通信，提供节点树浏览、组件编辑、记忆查看、世界操作等功能。

---

## 启动方式

```bash
# 方式一：通过 DevCli 打开
GameAgentDevCli inspect

# 方式二：直接在浏览器中打开
# 打开 tools/source/web/GameAgentCreator/index.html
```

---

## 界面布局

Creator 界面模仿主流游戏引擎的编辑器布局，分为以下区域：

- **顶部导航栏**：世界选择、语言切换、配置入口
- **左侧节点树**：按层级展示所有节点，支持收缩和拖拽
- **中央工作区**：显示选中节点的组件详情，支持键值对编辑
- **右侧检查器**：递归展示选中节点的所有数据（含属性和子对象）
- **底部监视器**：运行时信息、日志输出

所有子面板均可自由伸缩和滚动。

---

## 核心功能

### 节点树

- 按父-子层级展示世界节点
- 不同类型节点用不同颜色圆角标签标识
- 选中节点高亮，父节点路径浅色高亮
- 支持拖拽挂载子节点

### 工作区

- 查看和编辑选中节点的组件内容
- 组件名和操作按钮在顶部行展示
- KV 键值对递归展示全部嵌套层级
- 支持增、删、改键值对

### 检查器

- 递归展示节点属性的键值对
- 悬停弹窗展示完整内容
- 水平和垂直滚动条确保内容不超出屏幕

### 世界操作

- **创建世界**：在导航栏中创建新世界
- **导入配置**：从文件导入世界配置
- **复制世界**：在世界页面点击 "Clone World"，弹窗询问是否锁定源世界
- **世界设置**：修改 PipelineMode、记忆限制、传播参数等动态配置
- **推进 Tick**：推进世界时间线
- **运行自主行为**：触发节点自主行为
- **事件影响**：评估事件对世界的影响
- **局部推进**：推进指定范围的演化
- **重新规划**：重新生成世界大纲
