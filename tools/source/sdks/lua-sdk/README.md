# GameAgentEngine Lua SDK

This SDK is the Lua baseline client for GameAgentEngine.

## Current Scope

- baseline HTTP access layer
- health / version / invoke / runtime-task / callback entrypoints
- minimal integration examples

## Status

This directory currently provides a baseline scaffold, not full parity with the Go SDK yet.

## Notes

- intended as the lightweight script-bridge baseline
- focuses on URL construction and request-shape helpers first

## Included Examples

- `examples/health.lua`
- `examples/invoke_dialogue.lua`
