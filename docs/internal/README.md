# Internal Notes Index

This directory contains implementation notes, staged plans, deep-dive design records, and historical rollout documents.

It is intentionally separate from the public workflow, reference, and onboarding documentation.

## Recommended Reading Split

- Read `docs/` root, `guides/`, `reference/`, `integration/`, and `gameplay/` first for current user-facing workflows.
- Read `docs/internal/` only when you need deeper design context, migration history, or implementation rationale.
